# [HUB] Workshop PHP et Apache2

### 1. La machine
  Installez l'OS de votre choix sur une machine virtuelle. (VirtualBox ou VMware peu importe,
  cependant ce workshop est plutot fait pour VMware)
  Sur VMware configurez la connexion pour fonctionner avec Bridges.
  
  - Debian pour ne pas prendre de risque
  - CentOS/Redhat si vous etes un pirate
  - Ubuntu server si vous etes un guerrier. Attention car vous serez en tty.
